This week, we made three major improvements in our development methodology:

  • We began practicing version control in Github (branching and merging), as well each working out of our own individual Cloud9 workspaces
  • We practiced paired programming, with Bridgette and Ilan working together, and Chris and Cliff working together, respectively
  • We identified bugs in our code, created new issues in our product backlog, and then assigned and resolved each one, thereby passing more tests

Overall, we practiced better programming techniques and teamwork this week, and all became more comfortable using Github and Cloud9 appropriately. When we had questions regarding specific lines of code or story points that were not explicitly covered in the videos provided, we sought help in office hours, which was extremely helpful.

Likely due to our increased familiarity with Github and Cloud9, we were able to complete more story points in Sprint 2 than we had initially anticipated (see product backlog for more details).
